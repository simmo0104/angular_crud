# Angular Local Storage CRUD App

A simple, standalone Angular application demonstrating CRUD (Create, Read, Update, Delete) operations on employee data, persisted in the browser's localStorage. Built with Reactive Forms, validation, and Bootstrap for styling.

## Features
- Add new employees with form validation (required name, 6-digit pin code).
- View employee list in a responsive table.
- Edit existing records (updates all fields without duplication).
- Delete records with confirmation.
- Data persists across browser sessions.

## Tech Stack
- **Frontend**: Angular 17+ (standalone components)
- **Forms**: Reactive Forms with Validators
- **Styling**: Bootstrap 5
- **Storage**: Browser localStorage
- **Model**: TypeScript class for Employee

## Prerequisites
- Node.js (v18+)
- Angular CLI (`npm install -g @angular/cli`)
